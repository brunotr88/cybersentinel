**Cosa sono le "porte" di rete e perché dovresti preoccupartene?**

Immagina il tuo computer come un palazzo con tante porte.

Ogni porta serve a qualcosa:
- Porta 80: sito web
- Porta 443: sito web sicuro
- Porta 3389: Desktop Remoto Windows
- Porta 445: condivisione file Windows

Il problema? Alcune porte dovrebbero restare chiuse.

La porta 3389 (Desktop Remoto) è il bersaglio preferito dei ransomware. Se è aperta verso Internet, è questione di tempo prima che qualcuno provi a entrare.

La porta 445 (SMB) è stata la protagonista di WannaCry, l'attacco che nel 2017 ha paralizzato ospedali e aziende in tutto il mondo.
